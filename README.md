<strong>PORQUE EU GOSTO DE PYTHON</strong>

Python permite desenvolver scripts claros, previsíveis e fáceis de manter, ideais para rotinas críticas como validação de dados, automação de processos e análise de informações financeiras. Bibliotecas como pandas, numpy e pytest ampliam esse potencial, possibilitando tratamento de grandes volumes de dados, padronização de regras e controle de qualidade do código. O uso de SQL complementa esse ecossistema, garantindo acesso direto, estruturado e seguro aos dados, algo indispensável em ambientes onde integridade, rastreabilidade e histórico são fundamentais.

<strong>Ao focar nesse stack, priorizo:</strong><br>
-> Código estável em vez de complexidade desnecessária<br>
-> Dados corretos em vez de interfaces sofisticadas<br>
-> Processos automatizados em vez de tarefas manuais<br>

Esse foco reflete uma mentalidade alinhada ao mercado financeiro: menos improviso, mais controle; menos aparência, mais consistência.<br>
